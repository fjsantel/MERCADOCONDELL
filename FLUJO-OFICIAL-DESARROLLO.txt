FLUJO OFICIAL DE DESARROLLO WEB CON CMS GRATUITO
==============================================

🎯 OBJETIVO: Cliente paga SOLO dominio, sin hosting

FASE 1: DESARROLLO (DESARROLLADOR)
==================================
1. Crear repositorio GitHub del cliente
2. Desarrollar en VSCode con Claude Code
3. Configurar CMS (Netlify CMS)
4. Conectar GitHub → Netlify (solo para CMS)
5. Probar funcionamiento completo
6. Entregar acceso al cliente

FASE 2: DOMINIO Y DNS (CLIENTE/DESARROLLADOR)
============================================
1. Cliente compra dominio .cl (~$15.000/año)
2. Configurar Cloudflare (GRATIS):
   - Transferir DNS a Cloudflare
   - Configurar registros DNS:
     * CNAME www → usuario.github.io
     * A @ → IPs de GitHub Pages
   - Activar proxy (nube naranja)
3. En GitHub: Settings → Pages → Custom domain
4. Crear archivo CNAME en repositorio

FASE 3: FUNCIONAMIENTO FINAL
============================
✅ Cliente edita contenido en: sudominio.cl/admin/
✅ Cambios se guardan en: GitHub (automático)
✅ Sitio se actualiza en: sudominio.cl (GitHub Pages)
✅ Hosting: 100% GRATUITO
✅ Solo paga: Dominio ($15k/año)
